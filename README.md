# Zhili He - Personal Academic and Professional Website

## Overview

This repository hosts the personal academic and professional website of Zhili He, a researcher specializing in distributed systems, blockchain technology, and artificial intelligence. The website serves as a comprehensive platform to showcase academic achievements, research interests, professional experience, and technical projects.

## Features

- **Multilingual Support**: The website is available in six languages - Chinese, English, French, Japanese, Spanish, and German - to accommodate a global audience.
- **Responsive Design**: Optimized for viewing on various devices, ensuring accessibility across desktops, tablets, and mobile devices.
- **Academic Profile**: Detailed biography, research interests, and publication records.
- **Project Portfolio**: Extensive collection of research and development projects, including blockchain applications, AI systems, and software engineering endeavors.
- **Professional Experience**: Chronological overview of academic and professional milestones, including awards and recognitions.
- **Technical Skills**: Comprehensive listing of programming languages, frameworks, and tools proficiency.

## Structure

- `index.html`: Main Chinese version of the website.
- `index-en.html`: English version.
- `index-fr.html`: French version.

- `index-es.html`: Spanish version.
